# Webcam Clock Duel (2 players)

Малка уеб игра за двама пред една камера:
- Всеки има свой аналогов часовник.
- Случайно се генерира целево време (час + минути).
- **Дясна ръка** управлява часовата стрелка.
- **Лява ръка** управлява минутната стрелка (на стъпки от 5 минути).
- Лявата половина на кадъра е Играч 1, дясната е Играч 2.

## Стартиране

```bash
python3 -m http.server 4173
```

След това отвори `http://localhost:4173`.

## Технологии
- Vanilla HTML/CSS/JS
- MediaPipe Hands (CDN)
