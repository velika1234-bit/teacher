# Webcam Clock Duel (2 players)

Игра за двама пред една камера с **обща задача**:
- Показва се целево време (общо и за двамата).
- Всеки настройва своя аналогов часовник с ръце:
  - **дясна ръка** = часова стрелка
  - **лява ръка** = минутна стрелка (стъпка 5 мин)
- Който пръв улучи времето печели точка.
- Играта приключва, когато някой стигне **10 точки**.
- Има предварително подготвен pool от задачи и автоматично зареждане на следваща задача.

## Стартиране

```bash
python3 -m http.server 4173
```

Отвори `http://localhost:4173`.

## Технологии
- Vanilla HTML/CSS/JS
- MediaPipe Hands (CDN)
